# خطة نشر موقع "السجل المهاري"

## ملخص المشروع
- **نوع التطبيق**: Full-stack (Express + React)
- **قاعدة البيانات**: PostgreSQL
- **متطلبات الخادم**: Node.js 18+, 512MB RAM

---

## خيارات النشر المتاحة

### 1. Render.com (موصى به للمبتدئين)
- مجاني جزئياً
- يدعم PostgreSQL
- سهولة الإعداد

### 2. Railway
- مجاني جزئياً
- سهولة الإعداد
- يدعم Docker

### 3. VPS (خادم خاص)
- تحكم كامل
- يتطلب خبرة تقنية
- يمكن استخدام DigitalOcean, Linode, etc.

### 4. استضافة محلية (Docker)
- تشغيل محلي على جهازك

---

## خطوات النشر العامة

### الخطوة 1: إعداد قاعدة البيانات
- إنشاء حساب على مزود PostgreSQL (Neon, Supabase, Railway)
- الحصول على رابط الاتصال (Connection String)

### الخطوة 2: إعداد المتغيرات البيئية
- `DATABASE_URL` - رابط قاعدة البيانات
- `SESSION_SECRET` - مفتاح سري عشوائي

### الخطوة 3: بناء ونشر التطبيق
- `npm install`
- `npm run build`
- `npm start`
